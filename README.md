## Деплой проекта

Проект настроен на автоматическую сборку и публикацию через GitHub Actions. 
При каждом пуше в ветку `main`, билд отправляется в публичный репозиторий.

### Команда для публикации:
Для запуска автоматического деплоя достаточно выполнить:
git push origin main
