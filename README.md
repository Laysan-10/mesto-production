# Mesto

Интерактивный сервис для публикации фотографий с возможностью редактирования профиля и взаимодействия с карточками через API.

## Ссылки
* [Демо проекта](https://laysan-10.github.io/mesto-production/)

## Команды проекта

### Установка зависимостей
npm install

### Запуск в режиме разработки
npm run dev

### Сборка проекта
npm run build

### Публикация
Проект настроен на автоматический деплой через GitHub Actions. Для обновления версии на GitHub Pages достаточно выполнить пуш в основную ветку:
git push origin main
